# 🚚 Factory-to-Customer Shipping Route Efficiency Analysis for Nassau Candy Distributor

## 📌 Overview
This project analyzes factory-to-customer shipping routes for Nassau Candy Distributor to identify efficient routes, detect bottlenecks, compare shipping modes, and provide logistics insights through an interactive Streamlit dashboard.

## 🎯 Objectives
- Calculate Shipping Lead Time
- Analyze Factory → Customer routes
- Measure Route Efficiency
- Identify bottleneck regions
- Compare Ship Modes
- Build an interactive Streamlit dashboard
- Generate business recommendations

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Plotly
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## ▶️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Nassau-Candy-Shipping-Analysis.git
```

Move into the project:

```bash
cd Nassau-Candy-Shipping-Analysis
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
